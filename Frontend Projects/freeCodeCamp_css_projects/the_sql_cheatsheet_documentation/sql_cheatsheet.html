Introduction

SQL (Structured Query Language) is a standard language used for managing and manipulating relational databases. It allows users to store, retrieve, update, and delete data efficiently.

This cheatsheet provides a comprehensive guide to essential SQL commands with beginner-friendly explanations.

For more details, visit the official SQL documentation.

SELECT Statement

The SELECT statement is the most commonly used SQL command. It retrieves data from a database table.

  • Retrieve all columns:
    • SELECT * FROM customers;
  • Retrieve specific columns:
    • SELECT name, email FROM customers;
  • Filter results with WHERE:
    • SELECT name, email FROM customers WHERE age > 25 ORDER BY name ASC;
INSERT Statement

The INSERT statement is used to add new records to a database table.

  • Insert a new record:
    • INSERT INTO customers (name, email) VALUES ('John Doe', 'john@example.com');
UPDATE Statement

The UPDATE statement modifies existing records in a table.

  • Update a specific record:
    • UPDATE customers SET email = 'new@example.com' WHERE id = 1;
DELETE Statement

The DELETE statement removes records from a table.

  • Delete a specific record:
    • DELETE FROM customers WHERE id = 1;
JOIN Clauses

The JOIN clause is used to combine records from two or more tables.

  • Inner Join:
    • SELECT orders.id, customers.name FROM orders INNER JOIN customers ON orders.customer_id = customers.id;
GROUP BY

The GROUP BY statement groups rows with the same values into summary rows.

SELECT category, COUNT(*) FROM products GROUP BY category;
HAVING

The HAVING clause filters grouped data.

SELECT category, COUNT(*) FROM products GROUP BY category HAVING COUNT(*) > 10;
Indexes

Indexes improve query performance.

CREATE INDEX idx_customer_name ON customers (name);
Transactions

Transactions ensure data integrity.

BEGIN; UPDATE accounts SET balance = balance - 100 WHERE id = 1; UPDATE accounts SET balance = balance + 100 WHERE id = 2; COMMIT;
